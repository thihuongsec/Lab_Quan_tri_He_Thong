# Lab Quản trị hệ thống
_Quản trị an toàn hệ thống là quá trình lặp lại việc thiết lập, giám sát và duy trì các chính sách bảo mật trên toàn bộ hạ tầng công nghệ thông tin nhằm bảo vệ tài nguyên hệ thống khỏi các mối đe dọa và đảm bảo tính sẵn sàng, toàn vẹn, bảo mật của dữ liệu. Các bài lab dưới đây giới thiệu các phương pháp, công việc cơ bản để quản trị hệ thống một các an toàn_

1. Triển khai chính sách an toàn thông tin
2. Triển khai biện pháp truy cập tài nguyên an toàn
3. Triển khai mật mã trong bảo vệ an toàn truy cập tài nguyên
4. Triển khai an toàn cho dịch vụ web
5. Triển khai biện pháp đảm bảo khả năng sẵn sàng cho hệ thống

## Kết luận 
Chi tiết triển khai, mô hình triển khai và các bước phân tích được trình bày chi tiết trong file tài liệu .docx đã được tải lên repo này.
